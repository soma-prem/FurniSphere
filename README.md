Furnisphere is a web application clone of an e-commerce website focused on furniture. Users can browse products, explore items arranged by categories, and simulate placing orders. This project demonstrates the design and functionality of a furniture e-commerce platform without processing real transactions.

Features:

Browse furniture products by category
View product details
Simulate placing orders
Clean, user-friendly interface for exploring products
